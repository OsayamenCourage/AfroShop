AfroShop is a mobile-friendly e-commerce website built using HTML, CSS, and JavaScript for the frontend, with Node.js for the backend. It features product categories based on African-themed items, a shopping cart, user authentication, and multiple payment options.

Created in VSCode using:
- Live Server extension for preview
- Node.js for backend logic
- Express.js for routing
- MongoDB for data storage

To run:
1. Open VSCode and navigate to the AfroShop folder.
2. Run `npm install` to install dependencies.
3. Start the server with `node backend/server.js`.
4. Open `index.html` with Live Server to view the site.
